Welcome To The Terminal!

We will be using the Terminal app throughout the whole school year.

The Terminal lets you communicate with the Mac operating system with command lines. Using the Terminal and linking it with a website called GitHub, we will create and edit our websites!

    Three command lines that are essential in the Terminal:
  • ls - lists all the contents in the directory you are in
  • cd - changes the directory you are in
  • pwd - shows the pathway you came in the computer
    One example of using the three commands:
  1. In default, you start at the home directory.
  2. Run the ls command to see the directories you can access from the home directory. An example of the results would look like this:
    ls 
    Applications	Documents	Library		Music		Public
Desktop		Downloads	Movies		Pictures
  3. Let's go to the Document directory with the cd command:
    cd Documents
  4. And whenever you are lost, use the pwd command to show the path that you used:
    pwd
    /Users/studentID/Documents

There is a game called Terminus which is a good tool to get used to these three commands and other commands!
